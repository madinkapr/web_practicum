# Simple Login & Register Pages

This project demonstrates basic **Login** and **Register** pages built using pure HTML and CSS.

## Description

* Clean and minimal design for both login and registration forms.
* All styles are defined in the `app.css` file.
* Front-end only — no backend or authentication logic is included yet.

## Features

* Responsive layout using `meta viewport`.
* Well-styled input fields, buttons, and links.
* Gradient background and card-style container.

## Technologies

* HTML5
* CSS3

## Folder Structure

```
project-root/
├─ login.html       # Login page
├─ register.html    # Registration page
└─ app.css          # CSS stylesheet
```

## How to Run Locally

1. Place all files (`login.html`, `register.html`, `app.css`) in the same folder.
2. Open `login.html` or `register.html` in your browser (double-click or use “Open File”).

> Note: The `form` tag uses `action="#"`, meaning no backend is connected yet. Add a server later for real authentication.

## Possible Future Improvements

* Form validation (HTML5 + JavaScript) to display user input errors.
* Connect with backend (Node.js/Express, PHP, etc.).
* Add a show/hide password toggle.
* Store user data in a database (PostgreSQL, MongoDB, etc.).
* Improve UI/UX with animations and responsive layouts using media queries.

## Customization Tips

* Change the background color in `app.css` under the `body { background: ... }` rule.
* Adjust card styling in the `.login-container` class.
* Modify button color in the `.login-btn` class.
